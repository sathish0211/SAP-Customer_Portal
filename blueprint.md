# Project Blueprint

## Overview
This project is an Angular application for a Customer Portal. The current goal is to redesign the login page to feature a modern, "Antigravity" aesthetic with a split layout, 3D visuals, and a futuristic feel.

## Current State
- **Framework**: Angular v17+ (Standalone Components, Signals).
- **Existing Login**: Basic functional login page with simple styling.
- **Styles**: Global styles use Roboto font.

## Proposed Changes (Antigravity Login)
- **Layout**: Split screen (60% Left Visuals, 40% Right Login Form).
- **Visuals**:
    - **Left**: "Customer Portal" in futuristic typography, floating 3D shapes, neon gradients, antigravity animations.
    - **Right**: Glassmorphism login card, neumorphic inputs, gradient button.
- **Tech**: Pure CSS for animations and effects (performant and lightweight).
- **Typography**: Import 'Orbitron' or 'Exo 2' for headers.

## Plan
1.  **Global Styles**: Import futuristic fonts.
2.  **Login Component**:
    -   Refactor HTML for the split layout.
    -   Implement CSS for the "Antigravity" theme (deep space background, floating elements, glassmorphism).
    -   Ensure responsive design (stack on mobile).
3.  **Verification**: Build and visual check.
